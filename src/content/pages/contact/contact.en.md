# Contribute to the website

This website is always improving. Propose a change to the site by creating an account on Github and creating an issue [on this page](https://github.com/ateliers-velo/website/issues). If you prefer, you can talk about the issue on the #site-web channel in the Discord. You can also contribute directly to the website by creating a *pull request*.
